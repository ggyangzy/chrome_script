# Random Heart Move Effect

Chrome extension that shows floating hearts at random positions when the mouse moves.

## Included

- Manifest V3 extension
- English default UI
- Chrome icons: 16, 32, 48, 128
- Locales: English and Simplified Chinese

## Load Locally

1. Open `chrome://extensions/`
2. Enable Developer mode
3. Click `Load unpacked`
4. Select `/Users/gongjin/Personal/Coding/chrome`

## Package

Upload the generated zip file to Chrome Web Store after preparing the listing content and screenshots.
