Privacy Policy

Product: Random Heart Move Effect

Effective date: March 17, 2026

Random Heart Move Effect is a Chrome extension that displays floating heart animations on webpages when the user moves the mouse.

Data Collection

This extension does not collect, transmit, sell, or share personal data.

Local Storage

The extension stores only a simple preference in Chrome storage to remember whether the visual effect is enabled or disabled.

Webpage Access

The extension runs on webpages only to display the heart animation effect. It does not collect webpage content for analytics, advertising, profiling, or resale.

Third Parties

No user data is sold, transferred, or shared with third parties.
